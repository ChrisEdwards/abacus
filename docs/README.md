## Documentation Format

All documentation is written in **GitHub Flavored Markdown** and can be read:
- On GitHub (with nice rendering)
- In any text editor
- In your terminal (using `cat`, `less`, or markdown viewers like `glow`)

### Reading in Terminal

```bash
# Using cat (basic)
cat docs/getting-started.md

# Using less (scrollable)
less docs/getting-started.md

# Using glow (beautiful rendering)
glow docs/getting-started.md
```

## Keeping Documentation Updated

This documentation corresponds to the latest version of Abacus. If you're using an older version, some features may differ.

Check your version:
```bash
abacus --version
```
