## [Unreleased]

### Added

- GoReleaser configuration for multi-platform builds
- Release automation pipeline

## [0.1.0] - TBD

Initial release of abacus - a TUI viewer for Beads issue tracking.

### Added
- Interactive TUI for browsing Beads issues
- Issue list with filtering and sorting
- Detail panel showing full issue information
- Design and Acceptance sections in detail view
- Support for issue dependencies and relationships
- Auto-refresh capability
- Configuration file support
- JSON output mode
- Beads CLI version validation
